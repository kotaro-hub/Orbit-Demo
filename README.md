# Orbit-Demo

参考(TailWindCSS練習のため)にしたサイト -> https://orbit.kiwi/

## 使用言語
React、TypeScript、CSS（TailWindCSS）

## インストール
```sh
npm install
```

## サーバー起動
```sh
npm run dev
```
